## Clone the repository

```bash
git clone https://github.com/FazeelUsmani/PBA-AES-Activity-Group-7.git
```

### Open aes

```bash
cd aes
```

### Build the application

```bash
cargo build
```

### Run the application

```bash
cargo run
```

### Run Unit tests

```bash
cargo tests
```
